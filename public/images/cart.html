    bash
    python -m venv dialogflow-flask

    pip install flask

Step 3: Write the Flask Application

Here’s an example of a basic Flask app to handle Dialogflow requests:

python
from flask import Flask, request, jsonify

app = Flask(__name__)

# Webhook route for Dialogflow
@app.route('/webhook', methods=['POST'])
def webhook():
    # Parse the request from Dialogflow
    req = request.get_json(silent=True, force=True)
    intent_name = req['queryResult']['intent']['displayName']  # Get the intent name
    user_query = req['queryResult']['queryText']  # Get the user query

    # Handle intents
    if intent_name == "Order Tracking":
        response_text = "Please provide your order ID to track your package."
    elif intent_name == "Return Policy":
        response_text = "Our return policy allows returns within 30 days."
    else:
        response_text = "I'm sorry, I don't have an answer for that. Let me connect you with support."

    # Send response back to Dialogflow
    return jsonify({
        "fulfillmentText": response_text
    })

if __name__ == '__main__':
    app.run(debug=True)

  1. Run the Flask application:
    bash
    python app.py
  2. The Flask app will be running on http://127.0.0.1:5000/.
